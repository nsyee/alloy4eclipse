Abbreviated walkthrough (jogthrough)

In general, once you make a change to the model and want to see its effect on a run or check command,

  • save (ctrl+s)
  • build (ctrl+b)
  • select (from Commands menu)
  • execute and visualize (ctrl+e)

Remember, a solution is a counterexample to the claim you are checking (if you check and assert), or an example of the predicate your are running (of your run a pred).
