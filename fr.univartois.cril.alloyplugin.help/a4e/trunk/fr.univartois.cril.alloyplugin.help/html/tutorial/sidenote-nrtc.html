(non-reflexive) Transitive Closure (.^)

In Alloy, "foo.^bar" denoted the (non-reflexive) transitive closure of foo with respect to bar. It produces the set of everything you could produce by applying .bar" to foo one or more times. For example, the set includes 


  foo.bar
  foo.bar.bar
  foo.bar.bar.bar
  foo.bar.bar.bar.bar
  foo.bar.bar.bar.bar.bar
  ...
