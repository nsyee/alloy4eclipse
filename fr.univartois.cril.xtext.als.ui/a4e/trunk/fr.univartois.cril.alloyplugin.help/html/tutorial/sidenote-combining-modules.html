Combining modules with the open command

You can import a model named ord into your model by writing 

  open std/ord

Immediately below the module statement at the top of your model. In this case, std/ord is the path where ord is stored, relative to the "models" directory in the Alloy distribution. std is a folder containing standard packages provided with Alloy.

This is equivalent to prepending the contents of the ord model to your model. Consequently, you will have to be careful about name conflicts. In general, you will want to write your entire model in one module (one file). open is primarily for importing utilities, such at the ord module (which provides a way to do ordered state).
