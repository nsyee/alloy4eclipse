Alternate ways to run solvePuzzle

The following run command 

  run solvePuzzle for 4 Object, 8 State

could just as well have been written 

  run solvePuzzle for 4 but 8 State

or even 

  run solvePuzzle for 8 but 4 Object

In our case, however, the first of the 3 versions is probably the clearest, but one can imagine a case in which it would be much preferable to write 

  run solvePuzzle for 8 but 4 Object

rather than the longer, less clear, and more error prone 

  run solvePuzzle for 4 Object, 8 State, 8 foo,
      8 bar, 8 cuz, 8 quux, 8 fibbles, 8 thingy,
      8 majig, 8 floop, 8 whack, 8 blam, 8 pow
