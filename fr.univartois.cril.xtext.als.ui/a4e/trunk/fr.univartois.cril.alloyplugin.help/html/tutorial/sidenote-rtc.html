Reflexive Transitive Closure (.*)

In Alloy, "foo.*bar" denoted the reflexive transitive closure of foo with respect to bar. It produces the set of everything you could produce by applying .bar to foo zero or more times. For example, the set includes 


  foo
  foo.bar
  foo.bar.bar
  foo.bar.bar.bar
  foo.bar.bar.bar.bar
  ...
